Este proyecto corresponde a la clase incial del hibernate del Taller Java de la Universidad Nacional de la Matanza
Para este ejemplo se utilizo sqlite, un motor que guarda toda la info del esquema de base de datos
en un archivo binario. la ruta de este archivo binario es la que tienen que colocar en la 
propiedad connection.url del archivo hibernate.cfg. Dentro del proyecto el esquema en la carpeta src/main/resources, llamado hibernate.db.
A modo de ejemplo si al descomprimir el proyecto este queda en c:\\universidad\tallerjava\hibernate, el valor de la propiedad deberia se

c:\\universidad\tallerjava\hibernate\src\main\resources\hibernate.db
